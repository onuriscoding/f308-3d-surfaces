\subsubsection{Interpolation}

Dans sa forme la plus simple, l’interpolation linéaire (ou \textit{linear interpolation}) est définie par :
\begin{verbatim}
float lerp(float x, float y, float alpha):
    return (1 - alpha) * x + alpha * y
\end{verbatim}

L’interpolation se fait progressivement selon chaque axe (par exemple en 2D : d’abord selon $x$, puis selon $y$).
Le paramètre d’interpolation correspond à la partie fractionnaire des coordonnées du point dans la cellule.

Cependant, dans l’algorithme de Perlin, cette interpolation linéaire est combinée avec une autre fonction, celle de lissage.
La fonction de lissage (telle que la fonction \textit{fade}) est appliquée à ces paramètres avant interpolation, afin de garantir une transition douce entre les cellules et éviter les discontinuités visibles.
% mettre directement dans le Latex
\begin{verbatim}
float fade(float t) :
        return 6* t**5 - 15 * t**4 + 10 t**3
\end{verbatim}
